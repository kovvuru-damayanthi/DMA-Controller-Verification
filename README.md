# DMA-Controller-Verification
DMA Controller Verification using Verilog HDL and Synopsys VCS


## Overview
Verified a DMA (Direct Memory Access) Controller using Verilog HDL and Synopsys VCS.

## Tools Used
- Verilog HDL
- Synopsys VCS

## Verification
- Data transfer verification
- Control signal verification
- Address generation verification
- Interrupt verification
- Waveform analysis
